
## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #007bff;
    --secondary-color: #6c757d;
    /* ... */
}
```

### Content
- Update the HTML content in `index.html` to match your Figma design
- Adjust spacing, fonts, and layout in `styles.css`
- Add interactive features in `script.js`

## Matching Your Figma Design

To match your Figma design exactly:

1. **Colors**: Update CSS variables with colors from your Figma design
2. **Typography**: Adjust font families and sizes in `styles.css`
3. **Layout**: Modify the grid and flexbox layouts to match your design
4. **Components**: Add or remove sections based on your Figma frames
5. **Spacing**: Adjust padding and margins to match your design specs
